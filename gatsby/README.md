# Gatsby

## Note

Project bootstrapped with `gatsby new gatsby https://github.com/gatsbyjs/gatsby-starter-hello-world`

## Steps to Reproduce

1. Clone project
2. `cd gatsby`
3. `gatsby build`
4. If applicable, `npm install -g gatsby-cli`
5. `gatsby server`
6. http://localhost:9000
